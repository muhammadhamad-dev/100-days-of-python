# Day 04 — Rock Paper Scissors

A CLI implementation of Rock Paper Scissors against the computer.

## Concepts Practiced
- `random` module
- Lists and indexing
- `if/elif/else` with compound conditions
- Input validation

## How to Run
```bash
python rock_paper_scissors.py
```

## Example
```
Type 0 for Rock, 1 for Paper, 2 for Scissors
→ You chose Rock
→ Computer chose Scissors
→ You win!
```
