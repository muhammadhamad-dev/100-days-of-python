# Day 05 — Password Generator

Generates a random password with a user-defined mix of letters, symbols, and numbers.
Produces both an ordered (easy) and shuffled (hard) version.

## Concepts Practiced
- `for` loops
- `random.choice()` and `random.shuffle()`
- List to string conversion with `.join()`
- String concatenation

## How to Run
```bash
python password_generator.py
```

## Example
```
Letters: 4, Symbols: 2, Numbers: 2
Easy:  AbCd#!89
Hard:  #C8A!bd9
```
