# Day 02 — Tip Calculator

A CLI app that calculates how much each person pays when splitting a restaurant bill with tip.

## Concepts Practiced
- Type conversion (`float()`, `int()`)
- Arithmetic operators
- f-strings
- `round()`

## How to Run
```bash
python tip_calculator.py
```

## Example
```
Total bill: $150
Tip: 12%
People: 3
→ Each person should pay: $56.0
```
