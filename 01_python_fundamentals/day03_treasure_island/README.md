# Day 03 — Treasure Island

A text-based adventure game where choices determine the outcome.

## Concepts Practiced
- `if/elif/else` conditionals
- Nested conditions
- `.lower()` for input handling
- Multi-branch logic

## How to Run
```bash
python treasure_island.py
```

## How to Win
- Go `left`
- `wait` for the boat
- Choose the `yellow` door
