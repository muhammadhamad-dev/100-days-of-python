# Day 06 — Functions & Loops (Reeborg's World)

Day 6 was completed using Reeborg's World — a browser-based Python environment
for practicing functions and loops through robot movement challenges.

## Concepts Practiced
- Defining and calling functions (`def`)
- `while` loops
- `if` conditions inside loops
- Indentation rules
- Solving: Hurdle 1, Hurdle 2, Hurdle 3, Hurdle 4, Maze

## Environment
[Reeborg's World](https://reeborg.ca/reeborg.html)

## Note
No local project file for this day — exercises were run directly in the browser.
