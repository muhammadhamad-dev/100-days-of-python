# Day 07 — Hangman

A fully playable CLI Hangman game with ASCII art stages and a large word list.

## Concepts Practiced
- Importing from multiple modules
- `random.choice()`
- `while` loops with multiple exit conditions
- List indexing and updating
- Tracking game state

## Files
| File | Purpose |
|------|---------|
| `main.py` | Game logic |
| `hangman_art.py` | ASCII logo and hangman stages |
| `hangman_words.py` | Word list (180+ words) |

## How to Run
```bash
python main.py
```
