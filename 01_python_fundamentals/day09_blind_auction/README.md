# Day 09 — Blind Auction

A CLI blind auction program where multiple users submit secret bids.
After all bids are entered, the program reveals the highest bidder.

## Concepts Practiced
- Dictionaries (storing and retrieving key-value pairs)
- `max()` with `key` argument
- `while` loops
- Screen clearing with `print("\n" * 20)`

## How to Run
```bash
python main.py
```

## Example
```
Name: Ali     Bid: $150
Name: Sara    Bid: $200
Name: Hamad   Bid: $180

→ The highest bidder is Sara with Bid $200
```
